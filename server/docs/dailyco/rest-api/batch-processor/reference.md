# API endpoints

The [Batch Processor](/reference/batch-processor) API works currently uses a polling model; submit a job, poll for its status, fetch the output when jobs are finished. A push model may be available in the future.

  * [Submit a processor job](/reference/batch-processor/reference/submit-job)
  * [Get job status](/reference/batch-processor/reference/get-job)
  * [Get output from a job](/reference/batch-processor/reference/get-job-access-link)
  * [Get list of jobs](/reference/batch-processor/reference/list-jobs)
  * [Delete job](/reference/batch-processor/reference/delete-job)
