# DialOut

You can [start](/reference/rest-api/rooms/dialout/start), [stop](/reference/rest-api/rooms/dialout/stop) and [sendDTMF](/reference/rest-api/rooms/dialout/send-dtmf) for dialOut with the Daily API.

These REST endpoints are relatively similar to Daily's Client SDK APIs [`startDialOut()`](/reference/daily-js/instance-methods/start-dial-out), [`sendDTMF()`](/reference/daily-js/instance-methods/send-dtmf) and [`stopDialOut()`](/reference/daily-js/instance-methods/stop-dial-out) calls found in our [`daily-js`](/reference/daily-js) and [`daily-python`](https://reference-python.daily.co/api_reference.html#daily.CallClient.start_dialout) libraries.
