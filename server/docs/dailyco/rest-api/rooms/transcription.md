# Transcription

You can [start](/reference/rest-api/rooms/transcription/start) or [stop](/reference/rest-api/rooms/transcription/stop) a transcription with the Daily API when you have a credit card attached to your [Daily account](https://dashboard.daily.co).

The endpoints are mostly equivalent to the [`startTranscription()`](/reference/daily-js/instance-methods/start-transcription), [`updateTranscription()`](/reference/daily-js/instance-methods/update-transcription) and [`stopTranscription()`](/reference/daily-js/instance-methods/stop-transcription) calls found in our [`daily-js`](/reference/daily-js) library.

Read the [blog post tutorial](https://www.daily.co/blog/add-live-transcription-to-a-daily-call-with-our-newest-api) for detailed information on using transcription.
