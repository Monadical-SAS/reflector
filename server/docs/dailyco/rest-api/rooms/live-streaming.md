#

Live Streaming

[Pay-as-you-go](https://www.daily.co/pricing)

You can [start](/reference/rest-api/rooms/live-streaming/start), [update](/reference/rest-api/rooms/live-streaming/update), or [stop](/reference/rest-api/rooms/live-streaming/update) a live stream with the Daily API.

These endpoints are mostly equivalent to the [`startLiveStreaming()`](/reference/daily-js/instance-methods/start-live-streaming), [`updateLiveStreaming()`](/reference/daily-js/instance-methods/update-live-streaming), and [`stopLiveStreaming()`](/reference/daily-js/instance-methods/stop-live-streaming) calls found in our [`daily-js`](/reference/daily-js) library.

Head to our **[live streaming guide](/guides/products/live-streaming-recording/live-streaming)** for detailed information on using live streaming.
